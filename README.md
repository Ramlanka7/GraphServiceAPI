GraphService API

⸻

Overview

GraphService API is an enterprise-style backend service designed using clean architecture principles. It exposes a scalable GraphQL API with strong typing, clear separation of concerns, and reliable relational data management using PostgreSQL.

This project is intended as a foundation for building maintainable and extensible backend systems.

⸻

Tech Stack
	•	Node.js
	•	TypeScript
	•	Apollo Server
	•	GraphQL
	•	PostgreSQL
	•	Prisma ORM

⸻

Key Features
	•	Strongly typed backend using TypeScript
	•	GraphQL API with clear schema design
	•	Layered architecture (Resolver → Service → Repository)
	•	PostgreSQL for relational data integrity
	•	Scalable and maintainable project structure
